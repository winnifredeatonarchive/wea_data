When Do I Use This?

Use this utility page after you've created a new bibliography entry in the bibliography file. Note that to create a bibliography entry, you should do the following (but see the documentation for further instructions):

  1. Create a new <bibl> entry without an @xml:id. If this is an existing work, find the work's <listBibl> element; if it is not an existing work, then create a new listBibl element with the @xml:id for that work. For instance, if you discovered a new work called My Fair Lady, then you would create a new listBibl with an @xml:id of MyFairLady; if you were adding a new bibl for Pot of Paint, then you'd add a new bibl in the <listBibl> element with the @xml:id of PotOfPaint
  2. Once you've created and tagged the new <bibl>, it will be invalid and say that you need to give it an @xml:id. Click the red lightbulb on the left hand side beside the bibl, and press Create New Id.
  3. Once the bibl file is valid, go to this page to create a new document.

How To Use This

Once those bibls are added, go to this page and press the red Play button at the top of your oXygen screen (it's beside the red check mark button). It should take a few seconds.

Once they are valid, make sure to svn add those documents to the repository and commit your files.
